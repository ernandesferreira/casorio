Installation

VIA WordPress

  1. After your download log into your WordPress admin interface
  2. After login click on Plugins from the left  hand menu
  3. Click on “Add new”
  4. Click on “Upload”
  5. Browse to the directory you downloaded the plugin to and click “Install Now”
  6. After Wordpress has finished unpacking the file click on “Activate Plugin”
  7. After the plugin has been activated you will notice a new menu item on the left hand navigation labelled NEX-Forms
     

VIA FTP

  1. After your download unzip nex-forms from your download .zip
  2. Open your FTP client
  3. Browse to /your_wp_instalation/wp-content/plugins/ server
  4. Uplaod nex-forms into this directory
  5. Go to your wordpress admin panel
  6. Go to plugins and activate NEX-Forms
  7. After the plugin has been activated you will notice a new menu item on the left hand navigation labelled NEX-Forms
     

Calendar Manager

The calendar manager can be opened from the top left corner and opens automaticaly on page load.

Creating a calendar

  1. Enter a title.
  2. Enter a description. The description is for admin use.
  3. Click on Create

Using Calendars

VIA the Calendar Manager
  1. Open your calendar manager
  2. Click on the second last icon next to the calendar
  3. Read the instructions provided. And copy the code to be pated in the desired location
VIA Worpress TinyMCE
  1. Add/edit a page or post
  2. Make sure you are in "Visual" view and not "HTML" view.
  3. Click on the blue NEX-Forms icon found on the editor
  4. A popup will appear. Select the calendar to be inserted into the page/post
  5. Click inset into post and the shortcode will be generated.

Editing Calendars

  1. Open your calendar manager
  2. Click on the third icon
  3. Edit the calendar name and desciption
  4. Click Save.
The Calendar list item will show green for a few seconds after editing.

Adding an Event to a Calendar

Add new event

  1. Drag a single event or an icon to the desired date on the calendar. Please note...You need to save the event before changing month or year otherwise the event will be lost!
  2. Change the settings
      Event Info
    • Title - Sets the Event title
    • Description - Sets the Event description show in the modal box when clciking on the event
    • Popover Teaser Text - This will trigger a bootstrap popover when hovering over the event. This is just to give a very short summary of what the event is about.
      • Google Map
    • Vanue - Sets the venue name
    • Google Map - Add the address of the event location to add a google map
      • Styling
    • Event Icon - Sets a small icon in front of the event title and changes the icon is the case of icon events.
    • Event Color - Choose the bootstrap colors or select custom colors for text, background and text
    • Size - Set the size of the event as displayed in the calendar (x-small, small, normal, large)
  3. Click Save Event! (dont forget this)

Setting the event time

  1. After adding an event to a date click on the day number to opend the daily diplay.
  2. Times are set in 5min increments and the event can be dragged into any time slot. Note! The emty minute slots will not be visible on the front end, only the time slots containing an event and hour slots will be shown.
  3. Click Save Event!

Editing an event

  1. Hover over the event and click on the blue icon with the pencil. Please note to change the event to be on a different month or year, drop the event in the Clipboard and browes to the desired date and drag the event into the desired date.
  2. Change the settings
  3. Click Save Event!

Deleting an event

  1. Hover over the event and click on the red icon with the X.
  2. Click on delete permanantly when the confirmation box pops up or click cancel if you acidentaly clicked the delete bottom
  3. Click Save Event!
